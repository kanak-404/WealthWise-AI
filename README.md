# WealthWise AI – Predictive Financial Copilot

## 🚀 Problem

Most people track expenses but fail to control future spending due to lack of predictive insights.

## 💡 Solution

WealthWise AI predicts future financial behavior, explains spending patterns, and provides actionable advice using AI.

## 🔥 Features

* Smart expense tracking & categorization
* Spending prediction engine (ML-based)
* Behavioral insights (weekend spending, category trends)
* AI financial advisor (chat interface)
* Smart alerts for overspending

## 🛠 Tech Stack

* Backend: Flask
* ML: Scikit-learn
* Data: Pandas
* AI: OpenAI API (planned)
* Frontend: HTML/CSS/JS (planned)

## 🎯 Vision

To move from passive expense tracking to proactive financial decision-making.

## 🧠 Tagline

"Don’t just track money. Predict it. Control it."
